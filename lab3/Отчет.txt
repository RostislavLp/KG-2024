==================================Отчет=================================
Сравнение временных характеристик алгоритмов отрисовки линий


1.ЦДА:

 Временная сложность: O(n), где n - длина линии.
 использует дробные координаты, что может потребовать округления.
 Преимущества: простой в реализации, сравнительно быстрый.
 Недостатки: не очень точный, может быть проблематичным для больших расстояний.

2. Пошаговый алгоритм:

 Временная сложность: O(n), где n - длина линии.
  использует целочисленные координаты, шаг по оси с большим наклоном всегда равен 1.
 Преимущества: простой в реализации, точный для линий с малым наклоном.
 Недостатки: менее точен для линий с большим наклоном, может быть медленнее ЦДА для линий с большой длиной.

3. Алгоритм Брезенхейма:

Временная сложность: O(n), где n - длина линии.
 использует целочисленные координаты, шаг по оси с большим наклоном определяется с помощью ошибки.
Преимущества: точный, быстрый, может быть использован для отрисовки линий с большим наклоном.
Недостатки: более сложный в реализации, чем ЦДА и пошаговый алгоритм.

Заключение:

Алгоритм Брезенхейма является наиболее точным и эффективным из трех алгоритмов.
В лабароторной при изменении ширины сетки все поле очищается.
